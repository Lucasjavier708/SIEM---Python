# Lucas Pizarro | Cybersecurity Portfolio

**Portfolio profesional** de Lucas Javier Pizarro - Cybersecurity Analyst & SOC L1 Specialist.

## 🎯 Secciones

✅ **HERO** - Presentación con typing effect y animación de logo  
✅ **EXPERIENCIA** - Trayectoria profesional con tarjetas interactivas  
✅ **FORMACIÓN** - Educación formal y certificaciones  
✅ **STACK TÉCNICO** - Herramientas organizadas en 3 categorías alargadas  
✅ **PROYECTOS** - Casos de estudio (imagen cuadrada + tarjetas alargadas)  
✅ **LABORATORIO** - Descripción de infraestructura  
✅ **CONTACTO** - Links y CTA  

## 🎨 Diseño

- **Colores**: Dark mode neon (Cyan #00d9ff + Magenta #ff1493)
- **Tipografía**: IBM Plex Mono (código) + Inter (cuerpo)
- **Animaciones**: Binary particles (0 y 1 cayendo), matrix effects, typing, glow
- **Responsive**: Mobile-first, adaptable a todos los tamaños
- **Performance**: HTML5/CSS3/Vanilla JS (sin frameworks)

## 🚀 Cómo usar

1. Descargar archivo ZIP
2. Extraer en tu computadora
3. Abrir `index.html` en navegador
4. ¡Listo! 🌟

## 🔧 Customización

### Cambiar Colores

En `styles.css`, línea 8-14, modifica variables CSS.

### Agregar Herramientas al Stack

En `index.html`, busca `.stack-tools` y agrega:

```html
<div class="tool">
  <i class="fa-brands fa-tuherramienta"></i>
  <span>Herramienta</span>
</div>
```

### Cambiar Proyectos

En `index.html`, modifica `.proyecto-card` con `data-img` y contenido.

### Agregar Links Reales

Reemplaza todos los `href="#"` con URLs reales.
